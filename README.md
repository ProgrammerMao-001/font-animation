# bdg-map-lib

vue2版本的组件库模板

## 安装

使用 npm 安装 comp-demo

```bash
  npm install comp-demo
```

## 使用
下载源码直接二开即可，本组件库只是示例，不做任何功能
```bash 
git clone https://gitee.com/ProgrammerMao-001/compDemo.git
```
